# 開発メモ

## スクリプト実行
  npm startは、一般的に本番運用時にしか使わない。
  npm run devを開発時に使う。

npm installは、package.jsonがあるディレクトリ全てに対して必要。

## UI
  ヘッダー左端にメニューボタンを置き、ドロワー、ハンバーガーメニューを開く仕様にする。

## 次のステップ

1. **プロジェクト構造の設計**
   - バックエンド（Express）とフロントエンド（Vue.js）の分離
   - ディレクトリ構造の決定

2. **開発環境のセットアップ**
   - MongoDBのインストール・設定
   - Vue.jsプロジェクトの初期化
   - 必要なパッケージのインストール

3. **Figmaデザインの作成**
   - UIデザインの作成
   - デザインシステムの定義

4. **Phase 1の実装開始**
   - 認証機能の実装
   - 本の登録・一覧表示機能の実装
